# Task 7 – Identify and Remove Suspicious Browser Extensions

## Objective
The purpose of this task is to find unsafe or unnecessary browser extensions that may harm privacy and security, and then remove them.

## Browser Used
- Google Chrome (Version: 119.x)

## Steps Performed
1. Opened Chrome browser.
2. Went to Extensions page:
   - Menu (⋮) → More Tools → Extensions
3. Checked all installed extensions and their permissions.
4. Noted details like:
   - Is the extension useful?
   - Does it ask for risky permissions?
   - User reviews and rating on Web Store
5. Identified suspicious/unwanted extensions like:
   - "Wallpaper HD" → requested access to *Read browsing history*
   - "Free Shopping Helper" → unknown publisher
6. Removed both risky extensions using **Remove** button.
7. Restarted browser after removal.
8. Verified browser performance and privacy improvement.

## Suspicious Extensions Removed
| Extension Name | Issue | Action Taken |
|---------------|-------|--------------|
| Wallpaper HD  | Access to browsing history | Removed |
| Free Shopping Helper | Unknown publisher / Ads | Removed |

## What I Learned
- Always check permissions before installing any extension
- Extensions with access to passwords, history, clipboard = dangerous
- Unused extensions should be removed
- Malicious extensions can steal data or show ads
- Install only from trusted developers

## Best Practices
✔ Keep only necessary extensions  
✔ Check reviews and install count  
✔ Update extensions regularly  
✔ Use browser’s built-in security checker  
✔ Enable Safe Browsing feature  

## Conclusion
Managing browser extensions is important for cybersecurity.  
Removing suspicious ones improves security and privacy.
